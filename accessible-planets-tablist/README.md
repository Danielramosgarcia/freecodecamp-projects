# Accessible Planets Tablist

Aplicação de abas com informações sobre planetas, focada em acessibilidade.

## Funcionalidades

- Exibe informações sobre planetas (Terra, Saturno, Marte)
- Navegação por abas acessível com atributos ARIA
- Mostra/oculta conteúdo correspondente ao clicar na aba
- Apenas uma aba ativa por vez com indicação visual

## Tecnologias

- HTML
- CSS
- JavaScript
